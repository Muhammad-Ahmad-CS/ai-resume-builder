# 🤖 AI Resume Builder

An AI-powered resume generator web application built with **FastAPI**, **React.js**, and **Tailwind CSS**.

## 🛠️ Tech Stack
- **Backend:** Python FastAPI
- **Frontend:** React.js + Tailwind CSS
- **Database:** PostgreSQL
- **AI Integration:** OpenAI API

## 🚀 Key Features
- User Authentication (JWT)
- AI-based Professional Summary & Skill Generator
- Real-time Resume Preview & PDF Export
